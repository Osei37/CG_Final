# CG_Final

CGの授業の最終課題

processing を使った3次元CGアニメーション

**テーマ**

鳥居を通ることで時間を超えるアニメーション．最初は春の早朝だったのが，ワープすることで夏の昼，秋の夕方，冬の夜へと変化し，それに合わせて両脇の木も変化する．冬の夜の後，再びワープすると，別の木が生えた早朝の春から再び始まり，永遠にループする．
